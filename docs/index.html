CSS + ASS.js (WebGL) rendering engines. Auto-fetch from GitHub. Karaoke support. Per-style customization.

📦 Download ZIP 📖 Installation Guide

✨ Features

📂

Load .ass Files

From your computer or auto-fetch from GitHub repositories based on Video ID.

🎨

Dual Engines

CSS (HTML/CSS, default) or ASS.js (WebGL/Canvas accelerated). Toggle via header buttons.

🔍

Multi-Source

Add multiple GitHub repos as subtitle sources. Smart search with title-based relevance sorting.

🎤

Karaoke

Three-state karaoke (Pre/Active/Post) with syllable-level effects using \k tags.

⚙️

Per-Style Override

Customize position, size, outline, blur for each style individually. Full ASS Alignment + Margins support.

🔄

Auto Backup

Export/import all settings per video. Reset and restore with one click.

📦 Installation (Developer Mode)

1

Download or Clone

git clone https://github.com/zingky/YouTube-Aegisub-Loader.git

Or download ZIP and extract.

2

Open Chrome Extensions

Navigate to chrome://extensions/

3

Enable Developer Mode

Toggle Developer mode (top right corner).

4

Load Unpacked

Click Load unpacked → select the extension/ folder from the repo.

5

Enjoy 🎉

Visit any YouTube video → SUB button appears in the player controls.

⚠️ Ensure vnf-comic-sans.ttf is present in the extension/ folder.

🎯 Quick Usage

1. Load a .ass file

Click Choose File in the popup and select your .ass file. Or let it auto-fetch from GitHub.

2. Switch Engines

Toggle between CSS (default, recommended) and ASS.js (experimental, WebGL) using the buttons in the header bar.

3. Customize Styles

Adjust font size, outline, blur, colors globally or per-style via the STYLES panel on the right.

4. Manage Sources

Open the footer ⚙️ panel to add/remove GitHub subtitle sources. Configure backup & export.
